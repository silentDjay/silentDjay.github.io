CAN<i>WALK
Link to Project
My final project at the Iron Yard. A web app for reluctant walkers, CAN<i>WALK promotes walking by presenting personalized and relevant information on weather, physical geography, safety, accessibility, and enjoyability to prospective walkers. Our full stack team built an AngularJS front end app which heavily integrated the Google Maps JavaScript API and Places library, and a Rails app on the back end to store users' information related to walking history and places of interest.
Mapstery
Link to Project
A geography game for those who like a challenge. Inspired by my countless hours of playing map-based games on the web, Mapstery uses reverse geolocation via the Google Maps JavaScript API to match users' clicks to country names.
